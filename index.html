Esta página no tiene como objeto ser renderizada, sino tener registrado los atributos existentes. Por lo tanto, no me preocupé en si el código va a ser renderizado bien o no (lo más probable es que no)

Atributos Globales:

Fuente: mozilla

-autocapitalize: Controla si la entrada de texto se escribe en mayúsculas automáticamente y cómo lo hace cuando el usuario lo escribe o edita. Puede tener los siguientes valores:

  • off o none; no aplica autocapitalizacion
  • on o sentences: La primer letra de cada oracion es una mayúscula por defecto, todas las otras letras serían minusculas
  • words: igual que on/sentences
  • characters: todas las letras mayúsuclas

-class

-contenteditable: indica si el elemento podría ser editable por el usuario. En caso de que si, el navegador le permite al usuario editarlo. Tiene dos valores: .true o empty, el cual indica que es editable .false, el cual indica que el elemento no es editable

-dir: indica la direccionaliodad del texto (ltr, rtl, auto) de izquierda a derecha, derecha a izquierda

-draggable: Indica si un elemento puede ser arrastrado o no, es boolean (values true o false)

-hidden: Indica que el elemento aún no es relevante o ya no lo es. Por ejemplo, se puede usar para ocultar elementos de la página que no se pueden usar hasta que se haya completado el proceso de inicio de sesión. El navegador no renderizará dichos elementos. Este atributo no debe usarse para ocultar contenido que podría mostrarse legítimamente.

-id

-inert: Un valor booleano que hace que el navegador ignore los eventos de entrada del usuario para el elemento. Útil cuando hay eventos de clic presentes.

-itemid Identificador unico y global de un ítem

-lang: Idioma

-role: Indica el rol semántico de un elemento role="role_type"

-draggable: Indica si un elemento puede ser verificado en busca de errores ortográficos. Values: .empty o true (si lo es) .false (no lo es)

-style: Estilos embebidos

-virtualkeyboardpolicy: Controla el comportamiento de los teclados virtuales de dispositivos como tablets, celulares o cualquier otro dispositivo en el que no hay un teclado físico. Values: .auto/empty, el cual muestra el teclado cuando el elemento es tocado o seleccionado .manual, el cual desacopla el foco o toque desde el teclado virtual en el elemento
